Program D3 Manajemen Informatika dirancang untuk menghasilkan tenaga ahli madya yang kompeten dalam perancangan, pengembangan, dan pengelolaan sistem informasi berbasis web dan basis data.

Kurikulum mencakup pemrograman web (HTML, CSS, JavaScript, Bootstrap), basis data relasional, rekayasa perangkat lunak, serta praktik kerja lapangan pada semester akhir.

Lulusan dapat berkarier sebagai web developer, staf IT support, administrator basis data, atau analis sistem informasi pada berbagai instansi pemerintah maupun swasta.

Mata Kuliah Unggulan

Web Desain 2

HTML, CSS, JavaScript & Bootstrap untuk web responsif.

Basis Data

Perancangan & pengelolaan basis data relasional.

Rekayasa PL

Metodologi pengembangan perangkat lunak modern.

Informasi Tambahan

Setiap mahasiswa wajib menyelesaikan minimal 110 SKS termasuk PKL dan Tugas Akhir pada semester 5–6.
Web Desain 2

Membahas pembuatan halaman web interaktif menggunakan HTML, CSS, JavaScript, dan Bootstrap 4, termasuk penerapan plugin JavaScript dan prinsip Responsive Web Design.

  • SKS: 3 (2 Teori + 1 Praktik)
  • Semester: 4  |  Prasyarat: Web Desain 1
Basis Data

Mahasiswa belajar merancang skema basis data relasional, normalisasi, serta menulis query SQL untuk kebutuhan aplikasi sistem informasi.

  • SKS: 3 (2 Teori + 1 Praktik)
  • Semester: 3  |  Prasyarat: Pengantar TI
Rekayasa Perangkat Lunak

Membahas siklus hidup pengembangan perangkat lunak: analisis kebutuhan, perancangan, implementasi, hingga pengujian secara terstruktur.

  • SKS: 3 (2 Teori + 1 Praktik)
  • Semester: 5  |  Prasyarat: Basis Data
